One-Time Setup — JSONbin

JSONbin ek free service hai jo numbers ko cloud mein store karta hai — taki sabke devices par same numbers dikhein. Ek baar setup karo, phir hamesha easy update.

1

Free account banao

jsonbin.io par jaao → Sign Up → Email verify karo (free plan kaafi hai)

2

Naya Bin banao

Dashboard → "Create Bin" → Neeche diya JSON paste karo → Save karo:

{"totalFunds":150000000,"raised":4523750,"donors":312,"lastUpdated":"2026-05-25"}
3

Bin ko Public karo

Bin settings (gear icon) → "Private" toggle OFF karo → taaki main site bina key ke read kar sake

4

Bin ID aur Access Key copy karo

Bin ID URL mein hoga: jsonbin.io/v3/b/YAHAN_HAI
Access Key: Account → API Keys → Master Key copy karo

✓ Yeh fixed hai — change ki zaroorat nahi
Account → API Keys → Master Key. Sirf aapke browser mein save hoga.
